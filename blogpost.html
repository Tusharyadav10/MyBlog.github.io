How To Install MinGW

Author - Tushar Yadav
22 September 2023. 6 min read

First, you have to download here the MinGW package. Now go to the download folder and double click on the MinGW package, click on install then continue. Now it will automatically download its required files(Internet is required) , when completed click on continue. Now a window will appear, here you have to click on mingw32-base then mark it for installation. Similarly, do it for mingw32-gcc-g++. Now click on installation menu and then apply changes. A new window appear here click on Appply button and after installation click on close.




Your installation of the MinGW package is successfull. Now comes the setup part :

In window(cortana) search bar type Edit the system environment variables and on open button. A environment variables windows will open. Here select path and then click on edit button. A new window will open here click on New button then copy this C:\MinGW\bin and paste to the new column. Then press ok and again press ok. Finally the setup is done now you can install and software(Devcpp, Codeblocks, etc) and can enjoy coding. If you still faces some difficulty you can comment and I will try to give you solution asap.


Happy Coding!
